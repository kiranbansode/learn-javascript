# Primitives vs. Objects (Primitive vs. Reference Types)

| PRIMITIVES | OBJECTS        |
| ---------- | -------------- |
| Number     | Object literal |
| String     | Arrays         |
| Boolean    | Functions      |
| Undefined  | Many More...   |
| Null       |                |
| Symbol     |                |
| BigInt     |                |

1. **PRIMITIVES** : stored in **CALL_STACK**
2. **OBJECTS** : stored in _**HEAP**_
