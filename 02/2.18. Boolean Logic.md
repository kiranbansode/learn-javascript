# Boolean Logic

## AND &&

| AND   | TRUE  | FALSE |
| ----- | ----- | ----- |
| TRUE  | TRUE  | FALSE |
| FALSE | FALSE | FALSE |

## OR ||

| AND   | TRUE | FALSE |
| ----- | ---- | ----- |
| TRUE  | TRUE | TRUE  |
| FALSE | TRUE | FALSE |

## NOT !

Inverts(change) `true` to `false` and `false` to `true`
